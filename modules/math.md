# math module


## constants

* PI -> pi
* E -> euler's number
* G -> alias to gravity



## basic ones

* abs -> absolute value
* min -> minimum value
* max -> maximum value
* avg -> average value
* sin
* cos
* tan
* cotan
* power(number, power) -> raise to a power
* pow -> an alias to power
* floor
* round
* ceil
* sign -> it returns 0 if a number is 0, 1 is it's positive or -1 if it's negative
* sqrt
* trunc
