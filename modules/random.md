# random module


* random(a, b) -> chooses a random number between a and b. It works only with not decimal (not type) numeric values
* srandom(a, b) -> returns a smooth random number between a and b
* choose(...values or an array) -> chooses a random element and returns it
* rand() -> chooses a random smooth random between 0 and 1
* shuffle(..values or an array) -> randomly shuffles values and return them as an array
* chance(a percent (0-1)) -> returns true or false depending on whether it was successful
