# operators




## math operators

* + -> addition
* - -> substraction
* * -> multiplication
* / -> division
* % -> modulo operator
* ^ -> power (** can also be used as an alias to help others comming from other languages)



## comparision operators
* == -> equals
* != -> not equals
* > -> greater than
* < -> less than
* >= greater than or equal to
* <= less than or equal to

## boolean operators

* && -> and
* || -> or

## bitwise operators

* & -> and
* | -> or
* ~ -> not
* << -> left shift
* >> -> right shift
* <> -> xor

## math QUICK assignment operators

* += -> adds and assigns
* -= -> substracts and assigns 
* *= -> multiplies and assigns
* /= -> divides and assigns 
* %= -> uses modulo and assigns 
* &= -> uses and and assigns
* |= -> uses or and assigns
* ~= -> uses not and assigns
* <<= -> uses left shift and assigns
* >>= -> uses right shift and assigns
* <>= -> uses xor and assigns


## looter notation

syntax: a->b

allows you to get a random number between a and b

for example: 1->2 will give you either 1 or 2


## dice notation

### dA

it is equal to 1->A in looter notation. It basically throws a dice of A number of sides.

### BdA

it throws B times a dice of A sides.

### d

It is an alias for d6


## range

A..B will create a range starting at A and ending at B (including A and B). It can act as an array
