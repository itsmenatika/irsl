# modules

The code is divided into modules that can be reloaded dynamically and swapped.

The starting code will be always made into starting module named 'main'

modules can relay on each other.


## built-in modules

math -> for quick math functions



## using a module


to use a module you can use:
```
import math;
```
for already recognized modules or built-in ones


or
```
import "modulepath";
```
to manually search for modules
